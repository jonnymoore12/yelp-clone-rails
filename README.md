## Yelp Clone

A clone of the website Yelp, made during week 8 of Makers Academy using Ruby on Rails.

### How to use the app

 - Ensure you have Rails v 4.2.7.1 installed
 - $bin/rake db:create (create postgres database)
 - rails s (run rails server)
